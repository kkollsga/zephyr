Zephyr

A fast, GPU-accelerated text editor written entirely in Go. Buttery-smooth rendering meets precise syntax highlighting.

Download for your OS View on GitHub

Features

GPU-Accelerated

Every pixel rendered on the GPU via Gio. Smooth scrolling and instant response on every platform.

Tree-sitter Highlighting

Accurate, incremental parsing for 17+ languages including Go, Python, TypeScript, Rust, and more.

Cross-Platform

Native experience on macOS, Windows, and Linux. Platform-specific integration on each OS.

Code Folding

Collapse and expand blocks with color-coded line count indicators. Cmd+click for recursive fold.

Markdown Preview

Rendered markdown with code blocks, task list checkboxes, tables, and copy buttons.

Tabbed Editing

Chrome-style tabs with drag-to-reorder, overflow dropdown, and unsaved-changes indicators.

Themes

Dark and light themes configurable via YAML. Toggle with a single click in the tab bar.

Find & Replace

Inline search with regex and case-sensitive modes. Navigate matches with keyboard shortcuts.

Smart Editing

Auto-pairing brackets, language-aware indentation, soft-tab backspace, and intelligent undo.

After installing

macOS blocks apps that aren't signed with an Apple Developer ID. After dragging Zephyr to Applications, open Terminal and run:

xattr -cr /Applications/Zephyr.app

Then open Zephyr normally. You only need to do this once.

Download for Windows

Download Installer Portable ZIP

The installer adds a Start Menu entry, optional desktop shortcut, "Add to PATH", and "Open with Zephyr" context menu. Includes an uninstaller.

The portable ZIP requires no installation — extract and run zephyr.exe.

Download for Linux

AppImage .deb .rpm .tar.gz

AppImage — no install needed. Make executable and run:

chmod +x Zephyr-*.AppImage && ./Zephyr-*.AppImage

Debian/Ubuntu: sudo dpkg -i zephyr_*_amd64.deb

Fedora/RHEL: sudo rpm -i zephyr-*-1.x86_64.rpm
